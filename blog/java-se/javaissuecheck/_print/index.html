Java程序中获取当前进程的进程ID(Pid)

2020-02-02

Java如何获取Pid

private String getPid() {
		try {
			String jvmName = ManagementFactory.getRuntimeMXBean().getName();
			return jvmName.split("@")[0];
		}
		catch (Throwable ex) {
			return null;
		}
	}

上面的代码来着SpringBoot的ApplicationPid的代码。通过这段代码就可以获取到Pid

查看线程的CPU使用

2018-05-29

1 查看Java程序运行的进程号

jps
或者
top 命令查看
或者
ps -ef | grep java

2 查看Java进程下面的线程

top  -p pid -H

3 用jstack命令查询

jstack pid | grep -A 10 线程的Id

以上的截图都是基于阿里云的ESC服务器，和有些需要转换成16进制的有点不一样。大体的解决思路是一样的。分为三步：

  1. 获取需要关注的Java工程的运行PID
  2. 查看用top -p pid -HJava线程下面进程的情况，主要关注CPU的使用率
  3. 用jstack pid | grep -A 10 线程的Id 命令找到对应的在Java中的位置
